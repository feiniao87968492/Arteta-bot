# 爱憎分明好感度系统 - 设计说明

## 目标
改造好感度系统，使之涨跌幅度更大、更容易触发下降、机器人表现更爱憎分明。

## 改动范围
仅修改 `plugins/arteta_chat.py` 一个文件，涉及函数：
- `update_and_get_player()` — 重写好感度变动逻辑
- `process_chat()` — 增强 LLM prompt 的语气指令
- `handle_fav()` — 适配新的等级体系
- 好感度等级阈值调整

## 具体方案

### 1. 关键词三级分级检测

```
重度（-100 ~ -200）: 狗屎[球队]、傻逼[教练]、阿尔特塔滚、垃圾球队、解散吧、草泥马、CNM、废物教练、垃圾教练
中度（-40 ~ -100）: 下课、废物、垃圾、滚、解雇、退役、s b、SB、脑残、有毒、倒闭
轻度（-10 ~ -40）: 菜、不行、无语、哎、算了、没救、失望、摆烂
正面（+50 ~ +200）: COYG、加油阿森纳、好教练、信任、必胜、相信塔子、我们阿森纳是不可战胜的、冠军
正常（+10 ~ +50）: 无关键词命中时的普通发言
```

### 2. 好感度等级体系

| 等级 | 区间 | LLM 语气指令 |
|------|------|-------------|
| 看台内鬼 | < -50 | 愤怒、训斥、冷嘲热讽、回复最短 |
| 预备队 | -50 ~ -1 | 严肃警告、冷淡、审视 |
| 青训生 | 0 ~ 49 | 普通、公事公办 |
| 一线队 | 50 ~ 199 | 积极鼓励、认可努力 |
| 核心首发 | 200 ~ 499 | 热情褒奖、信任口吻 |
| 传奇队长 | ≥ 500 | 极度信任、兄弟感、长篇激励 |

### 3. LLM Prompt 增强

在 `process_chat()` 的 `final_prompt` 中新增三段式指令：

```
【信任度态势】当前 [nickname] 信任度 [fav]，等级 [level]。
本次变动：[fav_status]了 [abs(inc)] 点（原因：[reason]）。

【回复纪律】根据球员等级严格执行以下态度：
- 传奇队长：热情拥抱，用"兄弟""老伙计"称呼
- 核心首发：积极肯定，鼓励保持状态
- 一线队：正常沟通
- 青训生：简短回应，略带审视
- 预备队：言辞严肃，明确表达不满
- 看台内鬼：直接表达愤怒，用词严厉，只给最短回应
```

### 4. 变动原因字符串

传给 LLM 的原因包括：
- "发言中包含对球队/教练的恶意攻击"
- "发言中包含明显的负面言论"
- "发言中表达了轻微的不满"
- "正常积极交流" / "展现了对球队的热情支持"
