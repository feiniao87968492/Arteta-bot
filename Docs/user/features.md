# 阿森纳 QQ 群聊机器人 — 功能概览

基于 NoneBot2 + OneBot V11 的 QQ 群聊机器人，以阿森纳主教练米克尔·阿尔特塔的身份与群成员互动。

---

## 功能列表

### AI 对话
**指令：** `A` / `塔子` / `阿尔特塔` + 内容

以阿尔特塔的风格与群友聊天、聊球、讨论战术。接入 DeepSeek API，支持实时获取阿森纳比赛结果、英超积分榜、伤病信息、网络搜索等。

### 理科解题
**指令：** `算法` + 题目

调用 DeepSeek 模型解题，并使用 LaTeX 引擎渲染为图片输出，支持高等数学、算法、物理等复杂公式推导。

### AI 图片生成
**指令：** `画图` + 画面描述

通过 gpt-image-2 模型根据文字描述生成图片。

### 高级画图
**指令：** `画图-pro` / `画图pro` / `画图2` + 画面描述

使用独立 API endpoint 的高质量图片生成。

### 图生图
**指令：** `图生图` + 修改要求（需回复或附带图片）

基于参考图片进行二次生成，支持修改画面内容。

### 英超积分榜
**指令：** `英超局势` / `积分榜` / `英超排名`

从 football-data.org 获取实时积分榜数据，调用 AI 生成阿尔特塔风格的局势分析，渲染为图片输出。

### 好感度系统
**指令：** `好感度` / `档案` / `盒`

- `好感度` — 查看自己的信任度数值和队内定位
- `档案` [@某人] — 查看自己或队友的详细档案
- `盒` [@某人] — 查看队友的简明信息

基于 SQLite 持久化存储，记录群成员的活跃度和互动数据。

### QQ 名片赞
**指令：** `赞我` / `点赞我`

自动为群成员 QQ 名片点赞。普通成员每日 10 次，群管理员/群主/VIP 成员每日 50 次。

### 誓言系统
**指令：** `发誓` / `我的誓言`

- `发誓` [目标] — 立下誓言，记录在战术笔记中
- `我的誓言` — 查看自己立下的所有誓言
- `发誓 /CLEAR` — 清除所有誓言

### 球队管理（仅管理员）
**指令：** `下放` / `禁言` / `红牌` [@某人]

将违纪成员禁言 10 分钟并扣除信任度（需要机器人为群管理员）。

### 每日群聊总结
**自动发布：** 每天 22:30 定时发布

**指令：** `今日总结` / `日报`（管理员手动触发）

自动记录每日群聊记录，每晚以阿尔特塔的教练风格生成当日聊天总结，渲染为图片发送到群内。

### 阿森纳周报
**自动发布：** 每周一 09:00 自动发布

**指令：** `周报` / `weekly`（管理员手动触发）

自动爬取阿森纳最新新闻，生成阿尔特塔风格的周报并发布到所有群。

### 好感度排行
**指令：** `好感度排行` / `排行`

查看群内成员信任度排行榜。
