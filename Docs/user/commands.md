# 阿森纳 QQ 群聊机器人 — 指令表

| 指令 | 别名 | 说明 |
|------|------|------|
| `A` | `a`、`塔子`、`阿尔特塔` | AI 对话：以阿尔特塔风格聊天、聊球、讨论战术 |
| `算法` | `代码`、`leetcode`、`战术演练`、`算法题`、`amath`、`物理`、`数学`、`计算` | 理科解题：调用 AI 解题并渲染为图片 |
| `画图` | — | AI 图片生成：根据文字描述生成图片 |
| `画图-pro` | `画图pro`、`画图2` | 高级画图：使用独立 API 的高质量图片生成 |
| `图生图` | — | 基于参考图片二次生成（需回复或附带图片） |
| `英超局势` | `积分榜`、`英超排名`、`局势` | 英超积分榜：获取实时积分数据并生成局势分析 |
| `好感度` | — | 查看自己的信任度数值和队内定位 |
| `档案` | `profile`、`个人档案` | 查看自己或队友的详细档案（可 @ 某人） |
| `盒` | — | 查看队友的简明信息（可 @ 某人） |
| `好感度排行` | `排行`、`ranking`、`信任度排行` | 查看群内信任度排行榜 |
| `赞我` | `点赞我`、`like_me` | QQ 名片赞：自动为群成员点赞 |
| `发誓` | `我要发誓`、`立帖为证`、`备忘录` | 立誓：记录个人目标誓言 |
| `我的誓言` | `誓言记录`、`查看备忘录`、`查誓言` | 查看自己立下的所有誓言 |
| `发誓 /CLEAR` | — | 清除自己所有的誓言记录 |
| `帮助` | `help`、`menu`、`指令`、`命令` | 显示帮助菜单 |
| `今日总结` | `日报`、`daily` | 手动触发每日群聊总结（仅管理员） |
| `周报` | `weekly`、`weeklynews` | 手动触发阿森纳周报（仅管理员） |
| `下放` | `禁言`、`红牌` | 将违纪成员禁言 10 分钟并扣除信任度（仅管理员） |
| `塔闭嘴` | — | 让教练暂时闭嘴，不再响应 A/塔子/阿尔特塔 指令（仅当前群） |
| `塔说话` | — | 让教练恢复响应指令 |
| `刷新情报` | — | 刷新足球情报数据 |

---

### 自动任务（无需手动触发）

| 任务 | 时间 | 说明 |
|------|------|------|
| 每日群聊总结 | 每天 22:30 | 自动生成当日群聊总结并发送到群 |
| 阿森纳周报 | 每周一 09:00 | 自动爬取新闻生成周报并发布到所有群 |
