# 发誓/誓言系统

> 开发者文档：理解用户的誓言记录与管理机制。

## 1. 背景

誓言系统允许用户在对话中向机器人（阿尔特塔教练角色）立下誓言或目标，并进行追踪管理。用户可以通过指令创建誓言、查看自己的誓言记录、以及清除所有誓言。该系统旨在增加互动趣味性，让用户对自己的承诺负责。

---

## 2. 存储方式

誓言数据以 JSON 文件形式持久化存储，**不使用 SQLite 数据库**。

- 文件路径：`data/arteta_swears.json`（位于机器人根目录下的 `data` 文件夹）
- 数据结构：以用户 QQ 号为键，誓言列表为值的字典
- 读写方式：
  - `load_swears()` — 读取完整字典，文件不存在或损坏时返回空字典 `{}`
  - `save_swears(data)` — 将字典写回文件，使用 `ensure_ascii=False, indent=4` 格式化输出

```python
DATA_DIR = Path("data")
DATA_DIR.mkdir(exist_ok=True)
SWEARS_FILE = DATA_DIR / "arteta_swears.json"
```

## 3. 指令

| 指令 | 触发词 | 功能 |
|------|--------|------|
| `发誓 <目标>` | `发誓`、`我要发誓`、`立帖为证`、`备忘录` | 创建一个新的誓言/目标 |
| `发誓 /CLEAR` | `发誓 /clear`（不区分大小写） | 清除当前用户的所有誓言 |
| `我的誓言` | `我的誓言`、`誓言记录`、`查看备忘录`、`查誓言` | 列出当前用户的所有誓言记录 |

### 3.1 发誓 [目标]

使用 `swear_cmd` 处理器（优先级 5，阻塞模式）：

1. 提取用户消息中的纯文本内容。
2. 去除可能携带的命令前缀（`发誓`、`我要发誓`、`立帖为证`、`备忘录`）。
3. 若内容为空，提示用户输入目标并结束处理。
4. 若内容为 `/CLEAR`（不区分大小写），跳转清除逻辑。
5. 从 JSON 文件读取当前誓言数据。
6. 以用户 ID 为键，将 `{"time": current_time, "content": content}` 追加到对应列表中。
7. 保存回文件，返回教练风格的确认回复。

### 3.2 发誓 /CLEAR

清除当前用户的所有誓言：

1. 从 JSON 文件读取誓言数据。
2. 若用户 ID 不存在或其誓言列表为空，提示"没有誓言可清除"。
3. 否则删除该用户键，保存文件，返回确认清除的回复。

### 3.3 我的誓言

使用 `check_swear_cmd` 处理器（优先级 5，阻塞模式）：

1. 从 JSON 文件读取誓言数据。
2. 若用户 ID 不存在或其誓言列表为空，提示"还没有立过誓言"。
3. 否则遍历该用户的所有誓言，按 `[序号] 时间\n目标：内容\n` 格式拼接。
4. 返回完整的誓言列表。

## 4. 数据格式

```json
{
    "123456789": [
        {
            "content": "本赛季要进10个球",
            "time": "2026-05-10 14:30"
        },
        {
            "content": "每天加练射门30分钟",
            "time": "2026-05-10 15:00"
        }
    ],
    "987654321": [
        {
            "content": "不再迟到训练",
            "time": "2026-05-09 09:00"
        }
    ]
}
```

- `key`：用户 QQ 号（字符串形式）
- `value`：誓言列表，每个元素包含：
  - `content`（字符串）：誓言/目标的具体内容
  - `time`（字符串）：记录时间，格式为 `"%Y-%m-%d %H:%M"`（如 `2026-05-10 14:30`）
- 空文件或文件损坏时，按空字典 `{}` 处理

---

## 5. 每个用户最多存储的誓言数

当前代码**未设置硬性上限**。用户的誓言列表会随每次 `发誓` 指令不断追加，理论上无数量限制。

若后续需要控制存储上限，可在 `handle_swear()` 中 `all_swears[user_id].append()` 之前检查 `len(all_swears[user_id])`。例如：

```python
MAX_SWEARS_PER_USER = 50
if len(all_swears[user_id]) >= MAX_SWEARS_PER_USER:
    await swear_cmd.finish("你的战术笔记已经写满了！先履行已有的承诺再立新誓！")
    return
```
